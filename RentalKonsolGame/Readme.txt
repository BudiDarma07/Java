Pilihan Menu [1-5] : 

Pilih salah satu opsi misal 1,

--------------------
 Data Customer
--------------------
 1. Tambah Data
 2. Ubah Data 
 3. Hapus Data
 4. Lihat Data
 5. Kembali
--------------------
Pilihan Menu [1-5] : 

Pilih salah satu opsi misal tambah data, maka akan ada form pengisian data seperti id customer, nama, dan lainnya
jika memilih ubah data maka akan mengubah data yang telah diinput sebelumnya misal 1. maka data customer yang berada pada list 1 yang akan diubah
jika memilih hapus data sama seperti ubah data
dan jika memilih lihat data maka dapat menampilkan semua data customer yang telah diinput atau sebelumnya

Pilih menu Kembali maka akan kembali kehalaman utama sistem
Untuk opsi data penyewaan akan sama seperti data customer
Selanjutnya pilih opsi Data penyewaan sistem akan menampilkan seperti berikut
Data Penyewaan
--------------------
 1. Tambah Data
 2. Ubah Data 
 3. Hapus Data
 4. Lihat Data
 5. Kembali
--------------------
Pilihan Menu [1-5] :

pada menu ini harus dimulai dari tambah data karena belum ada data yang tersimpan. user harus memasukan id penyewaan, id customer, konsol, dan metode pembayaran
user harus memasukan id customer karena id customer menjadi kunci untuk memanggil data customer dan status penyewaan dari customer

selanjutnya untuk opsi lihat data, sistem akan menampilkan data penyewaan yang berupa status penyewaan dan data customer
